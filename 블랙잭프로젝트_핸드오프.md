## 3. 투자 논리 (확신의 근거)

확신 위계: **AI 슈퍼사이클 → 메모리 → 레버리지**. 미국장 > 국장, 하이닉스 > 삼성.

- **공급 부족 2027년까지 지속**: 새 capacity는 fiscal 2028까지 의미 없음. 팹 증설 18~24개월. AI가 메모리 공급 70% 소비.
- **이익이 주가보다 빨리 늠**: 마이크론 EPS $1.56→$12.20→~$19. Forward PER 11~16배(쌈). 하이닉스 7배.
- **HBF가 다음 촉매**: 샌디 선두, 하이닉스 협업, 마이크론 개발 중.
- 하이닉스 본주·하닉레버·ADR은 같은 회사라 연동 → 미국서 오르면 하닉레버도 오름. ADR로 옮길 필요 없음.

## 4. 레버 본질 (검증한 것 — 흔들릴 때 기억)

- **decay**: 횡보장의 무덤. 출렁이면 녹음. 모닝스타 2009~2018 2x 레버 연 -11.1% vs 지수 +15.7%. 한 방향이면 폭발(TQQQ 연 51%).
- **서킷브레이커**: 하루 -50% 불가능. ±10% 정지, 시장 -7/-13/-20%. 진짜 -70~90%는 몇 달 추세하락+decay 누적(SOXL 2022 -70%, TQQQ -71%).
- **오른%≠빠진%**: +50% 후 -50% = 원금 -25%. 본전은 -33%. 그래서 고점 대비 낙폭이 기준.
- **청산 없지만 회복 불가가 무섭다**: 펀드라 마진콜 없음. -50%→+100%, -70%→+233%, -90%→+900% 필요. -50%는 회복 불가 가기 전 마지막 결단선.
- **단타는 처음부터 끝까지 모른다 (확신)**: 비트에서 직접 겪음(예수금 전 260만 당겨 샀고 그날 떨어져 후회했는데 다음날 +20%). 고점/저점 그 순간엔 절대 모름. "나는 단타 못하는 사람 = 장기로 묻어야 하는 사람"이라는 데이터.

## 5. 신호등 시스템 (teber-track) — 완성됨

### 배포 정보
- GitHub repo: **teddyberryarch/teber-track** (Public)
- GitHub Pages: https://teddyberryarch.github.io/teber-track/
- Secrets 설정됨: TG_TOKEN(BotFather), TG_CHAT_ID=7958777842
- GitHub Actions cron (하루 2회), 비용 0, 자동주문 없음

### 신호등이 보는 것 (5가지)
1. **🚀 이정표** — 시작 평가액(1억 4,190만) 대비 +30% 복리. 1단계=1억 8,447만, 2단계=2억 3,981만. 도달 시 방어선도 상향.
2. **🟡🟠🔴 방어선** — 고점 대비 -25%(1억 642만)/-40%(8,514만)/-50%(7,095만). 한 번 놓쳐도 다음 단계 또 알림. + 원금 하한 6,000만 병행.
3. **🔴 본주 레드플래그** — 하닉·마이크론·샌디 **본주가 고점 대비 -30%** 빠지면 알림. **등락률은 화면에 표시 안 함**(싱숭생숭 방지), 알림만.
4. **🟡 횡보** — 고점 후 3개월(60거래일) 정체 시 decay 점검.
5. **heartbeat** — 주 1회 "봇 정상" 신호. 안 오면 봇 멈춤 = 확인.

### 화면 구성
- 큰 신호등 orb + 상태
- 이정표 (단계 + 진행 막대 + "다음 1.84억 도달 시")
- 3단 방어선 (각 금액 표시)
- 횡보 카드
- 보유 4종목 (HBM/HBF/LOGIC 색태그)
- **점검 질문 3개 복붙 버튼** (HBM 비중 / 계약가 스프레드 / 하이퍼스케일러 capex → Claude에 물어보기용)
- **실적·이벤트 캘린더** (6/24 마이크론~12월, 1년치)
- 레드플래그 체크리스트 (감산/가격하락/capex둔화/삼성HBF/공급급증)
- 초심 + thesis 링크

### 평가액은 숨김
현재 정확한 평가액은 일부러 안 보여줌(매일 흔들리면 1년 못 감). 목표선(이정표·방어선) 금액만 표시.

### 점검 ≠ 매도
방어선 울려도 자동 손절 아님. **전제(슈퍼사이클) 깨졌나만 확인.** 조정 -20~40%는 예상된 것, 안 던짐. 레드플래그 3개+ 공식 발표로 겹칠 때만 탈출.

## 9. 현금흐름 / 생활 (미해결 고민)

- 월 고정지출 100~200만, 수입 "아예 불확실"
- 핵심 통찰: 투자보다 **수입원 확정**이 먼저. 로마안경원 등으로 월 수입 만들면 봉인이 안전해짐 (돈 떨어져서 주식 볼 일 없게)
- "빨리 사업해야 한다" = 당장은 로마안경원으로 런웨이, 베러비전은 그 위에서 장기로
- 곧 쓸 돈은 투자 X (단타라서). 전 재산 다 넣었으니 수입원 확보가 시급.

## 10. 톤 / 맥락 (다음 채팅이 알아야 할 것)

- 한국어 반말. 양지근은 디테일·현실 중시 (비현실적 극단 예시 싫어함, "현실로 말하자").
- 자기인식 정확: "단타 못한다", "고점에 산다"를 앎 → 그래서 장기 묻기. 머리(이해)가 손가락(행동)보다 앞섬 → 그래서 강제 장치(신호등) 필요.
- 확신 있으면 크게 거는 스타일. 분산 권유보다 확신 검증을 도와주는 역할.
- "그 사람(눈 돌아간 투자자) 될까봐" 걱정함 → 그 걱정하는 것 자체가 안전판. 반증 통로(신호등+사람) 열어두되 거르는 것도 본인 판단.
- zip 말고 파일 따로 줄 것.
- 큰 글씨는 줄이고 작은 글씨를 키울 것 (작은 게 안 보이는 거지 큰 게 작은 게 아님).
